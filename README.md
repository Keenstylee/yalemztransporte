Este proyecto es una página web diseñada para Yalemz Transporte y Logística, enfocada en brindar una experiencia clara, moderna y responsiva a los usuarios que buscan información sobre destinos, servicios y opciones de transporte.

El sitio está desarrollado con HTML, CSS y JavaScript, empleando buenas prácticas de diseño, validaciones, animaciones suaves y una estructura optimizada para dispositivos móviles y computadoras.
